Module ORGM.Server.Callbacks

Callback Functions.

All events are hooked in ORGMServerEvents.lua and call these functions.

It is unlikely that you will need to call any of these functions manually.

clientCommand (module, command, player, args)
Handles command requests from the clients in MP.

Triggered by the OnClientCommand event.

If the module name is ‘orgm’ and the ORGM.Server.CommandHandler has a function with the name that matches the command string argument, it calls that function.

Parameters:

  • module string name of the command module.
  • command string name of the command to run.
  • player IsoPlayer player that sent the command.
  • args variable data to be passed onto the command call.
