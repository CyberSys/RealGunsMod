Methods
addModel (name[, model[, texture]])
Loads a 3d model. Trigged by OnGameBoot event in ORGMClientEvents.lua

Parameters:

  • name string the name alias for the model
  • model string the file name. defaults to the name argument if nil. (optional)
  • texture string the texture file name. defaults to the name argument if nil. (optional)

unequipItemNow (player, item)
Instantly unequip the item if it’s in the player’s primary hand, skipping timed actions.

Used by Client.getFirearmNeedsUpdate() above when upgrading weapons to new ORGM versions.

Parameters:

  • player IsoPlayer
  • item HandWeapon
